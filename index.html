Personalized learning path by estimating student knowledge level.


Methodology:  Knowledge Space Theory takes a map of what skills lead to other skills and uses data to figure out where on the map a specific student is. 

References:
Doignon, J. P., Falmagne, J. C. (2015). Knowledge Spaces and Learning Spaces, arXiv:1511.06757

The idea for estimating student's knowledge state can be Visualized from this diagram and we are taking the example of student's knowledge on fractions.

1) Identify Skills in Fractions

We will give certain brain storming questions to the students to identify their comfort level in a certain topic( here the topic is Fractions).

2) Map Skill Relationships

Assumption: If a student knows how to find equivalent fraction (node c), he will most likely be able to identify a fraction (node a).
(for a relation to be valid, P(parent / child) > 0.75 and P(child / parent) < 0.50 should hold) 

3) Administer diagnostic test

Two actual students are attempting the adaptive diagnostic test.

Diagnostic algorithm selects the question with the maximum information gain - 'e'.

Jane attempts it correct while John gets this wrong.

Jane knows 'e' and all the parent skills of 'e' ('a', 'c', 'b').
John doesn't know 'e' and all the child skills of 'e' ('h').

For Jane, 'i' gets selected next.
For John, 'c' gets selected next.

Jane's question: Diya had 3/4m of a cloth piece. She used 1/2m for her doll's dress. How much cloth is left with her now?
John's question: An equivalent fraction of 2/3 is?

Jane knows skill 'i'. Hence, we can infer she
knows skill 'g' and 'd' as well.

John knows skill 'c'. Hence, we can infer he knows skill 'a' as well.

For Jane, 'h' gets selected next.
For John, 'd' gets selected next.

Jane gets 'h' question wrong.
John gets it right.

Jane doesn't know skill 'h'. Test completed for Jane.
John knows skill 'd'. No additional skill could be inferred at this moment.

'g' gets selected next for John.

John gets this wrong.

John doesn't know 'g' and all the child skills of 'g' ('i').

'b' gets selected next for John.

John gets this correct.

John knows skill 'b'. Test completed for John.

Test completed for both Jane and John.

4) Recommend Learning Path  

Based on the test result, we will decide proper learning path for Jane and John.
